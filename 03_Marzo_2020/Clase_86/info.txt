1) React Fragment, sirve para poner mas de un elemento consecutivo. Ya que en React solo puede haber un elemento padre.
Ej:
Sin Fragment, esto da error:
    return (
        <div></div>
        <span></span>
    )

Con Fragment:
    return (
        <>
            <div></div>
            <span></span>
        </>
    )
