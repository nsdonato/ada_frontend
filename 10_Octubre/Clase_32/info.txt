Operadores logicos.
&& (and) -> va a ser verdadera, si ambas partes son true
|| (or) -> nos va a devolver true, si alguna es true.

true && true // true
true && false // false
false && false // false
false && true // false

true || false // true
false || true // true
true || true // true
false || false // false.

!true // false
!false // true
